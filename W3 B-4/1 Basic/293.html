Horizontal Direction

Use .flex-row to make the flex items appear side by side (default).

Use .flex-row-reverse to right-align the direction, and use .flex-sm|md|lg|xl-row-reverse to right-align the direction on a specific screen size. Resize the browser window to see the effect.

flex-row-reverse:

Flex item 1
Flex item 2
Flex item 3

flex-sm-row-reverse:

Flex item 1
Flex item 2
Flex item 3

flex-md-row-reverse:

Flex item 1
Flex item 2
Flex item 3

flex-lg-row-reverse:

Flex item 1
Flex item 2
Flex item 3

flex-xl-row-reverse:

Flex item 1
Flex item 2
Flex item 3
